### 문제 설명

<p>태수는 25년간의 오랜 노력 끝에 알파벳 A, B, C를 익혔다. 인생의 최대 성과를 낸 태수는 이를 기념하는 놀이를 만들었다. 그 놀이는 빈 문자열에서 시작해서 A, B, C로만 이루어진 문자열 <em>S</em>를 만드는 놀이이다. 태수는 1초 동안 아래의 작업 중 하나를 수행할 수 있다.</p>

<ul>
	<li>문자열의 임의의 위치에 <code>A</code>를 추가한다.</li>
	<li>문자열의 임의의 위치에 <code>B</code>를 추가한다.</li>
	<li>문자열의 임의의 위치에 <code>C</code>를 추가한다.</li>
	<li>문자열의 임의의 위치에 <code>ABC</code>를 추가한다.</li>
</ul>

<p>태수가 문자열 <em>S</em>를 만드는데 걸리는 최소 시간을 구해서 좀 더 빠르게 놀이를 마칠 수 있도록 도와주자!</p>

### 입력 

 <p>첫째 줄에 <code>A</code>, <code>B</code>, <code>C</code>로 이루어진 문자열 <em>S</em>가 주어진다. (1 ≤ |<em>S</em>| ≤ 500)</p>

### 출력 

 <p>문자열 <em>S</em>를 만드는 데에 걸리는 최소 시간을 출력한다.</p>
